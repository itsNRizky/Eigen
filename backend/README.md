# How to Run Locally

Clone the project

```bash
  git clone https://github.com/itsNRizky/Eigen.git
```

Go to the project directory

```bash
  cd Eigen/backend
```

Install dependencies

```bash
  npm install
```

Setup .env

```bash
  DATABASE_URL=""
  PORT=
```

Setup Prisma

```bash
  npm run prisma
```

Add Seeds Mock Data

```bash
  npm run seeds
```

Run Dev Environment

```bash
  npm run dev
```
